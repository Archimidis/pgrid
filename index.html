Pgrid

P-Grid, a self-organizing structured peer-to-peer system.

Download this project as a .zip file Download this project as a tar.gz file

Middleware for Fault Tolerant PGrid Networks

P-Grid is a self-organizing structured peer-to-peer system, which can accommodate arbitrary key distributions, still providing storage load-balancing and efficient search by using randomized routing. This project is an implementation of the peer-to-peer infrastructure of P-Grid. The main goal is to add a fault-tolerant protocol. The implementation is based on SOA principles. This project is part of my dissertation.

Getting started

Libraries in use

  • Google guice 3.0
  • logback 1.0
  • slf4j 1.6.4

Compilation

Run the following commands to produce all the needed corba interfaces and classes before compilation. 1. cd pgrid 2. idlj -fall -td src/ resources/pgrid_corba.idl

Finally run the ant script.
